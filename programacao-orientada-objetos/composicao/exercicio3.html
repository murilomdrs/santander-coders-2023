Exercício: Modelagem de Veículos

Crie duas classes em JavaScript: "Veiculo" e "Garagem".

Classe Veiculo:

  • Deve ter as seguintes propriedades:
    • 'marca' (uma string que representa a marca do veículo).
    • 'modelo' (uma string que representa o modelo do veículo).
    • 'ano' (um número inteiro que representa o ano de fabricação do veículo).
  • Deve ter um método 'informacoes()' que retorna uma string com as informações completas do veículo (marca, modelo e ano).

Classe Garagem:

  • Deve ter uma propriedade veiculos (um array) para armazenar os veículos na garagem.
  • Deve ter um método adicionarVeiculo(veiculo) que permite adicionar um veículo à garagem.
  • Deve ter um método listarVeiculos() que retorna um array com as informações de todos os veículos na garagem.

Seu objetivo neste exercício é criar as classes "Veiculo" e "Garagem" e implementar os métodos necessários para adicionar veículos à garagem e listar os veículos presentes nela. Isso lhe dará a oportunidade de praticar a modelagem de classes e composição de objetos.
